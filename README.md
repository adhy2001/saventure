# Saventure Portfolio

A responsive portfolio website for Saventure, a creative agency specializing in digital transformation and innovative marketing solutions.

## Features

- Responsive design optimized for all devices
- Contact form with EmailJS integration
- Modern UI with smooth animations
- Mobile-first approach

## Technologies Used

- HTML5
- CSS3
- JavaScript
- Bootstrap 5
- EmailJS for contact form

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. For development, serve the files through a local server

## Contact

For inquiries, please use the contact form on the website or reach out via WhatsApp.
